express, mongodb로 게시판 만들어보기 ! -> 책 응용하여 나중에 나만의 프로젝트 만들어보자

프로젝트 구조 : MVC 모델

게시판 프로젝트의 데이터 흐름

VIEW(웹페이지) <--> Express-Router(컨트롤러) <--> 서비스(서비스함수/클래스) <--> 데이터 액세스(몽고디비 모델)
                                                        ⬆️
                                                    모델(데이터 모델)

(1) 뷰 열할을 하는 웹페이지에서 익스프레스로 요청
(2) 받은 요청에 대한 인증, 유효성 검증 후 서비스 함수 호출
(3) 컨트롤러에서 받은 데이터로 로직 처리/ 데이터 엑세스로 넘기는 모델 객체 생성
(4) 서비스 계층에서 받은 데이터를 CRUD 수행
(5) 결과를 다시 서비스로 넘김
(6) 받은 데이터 결과를 컨트롤러로 넘김
(7) 웹페이지에 응답으로 보냄

View 디렉토리 : 뷰 관련 파일
Controllers 디렉토리 : 컨트롤러 관련 파일
Services 디렉토리 : 서비스 관련 파일
Config 디렉토리 : 설정 관련 파일
